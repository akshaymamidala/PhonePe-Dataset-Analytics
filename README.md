# PhonePe Analytics Dashboard | Power BI + Figma

An interactive business intelligence dashboard for **PhonePe** — India's leading digital payments platform. Designed with pixel-perfect UI in Figma and built in Power BI.

![PhonePe Dashboard](https://github.com/akshaymamidala/PhonePe-Dataset-Analytics/blob/main/Final_Dashboard.jpg)

---

## 🎯 Project Overview

This dashboard provides deep insights into PhonePe’s transaction performance, user behavior, and business metrics with a modern, brand-aligned purple theme.

### Key Metrics
- **₹3.47 Billion** – Total Transaction Value
- **300K** – Total Transactions
- **96.00%** – Transaction Success Rate
- **108K** – Total Active Users

---

## ✨ Features

- Monthly trends in transaction value and volume
- Service-wise performance (Loans, Insurance, Money Transfer, Recharge, etc.)
- Weekday vs Weekend analysis
- Age group segmentation (Gen Z, Millennials, Gen X, Boomers)
- Month-over-Month (MoM) growth comparison
- Payment success vs failure breakdown
- Fully interactive filters and slicers

---

## 🛠 Tech Stack

- **Power BI** – Data modeling, DAX, visualizations & interactivity
- **Figma** – UI/UX Design (Pixel-perfect PhonePe branding)
- **DAX** – Custom measures and calculations

---

## 📂 How to Open & View the Dashboard

### Prerequisites
- Install **Microsoft Power BI Desktop** (Free) → [Download here](https://powerbi.microsoft.com/desktop/)

### Steps to Open:

1. Clone or download this repository:
   ```bash
   git clone https://github.com/yourusername/phonepe-dashboard.git
   ```

Open the folder and locate the file:
PhonePe.pbix

Double-click PhonePe.pbix to open it in Power BI Desktop.
Once opened:
Go to View → Enable "Fit to Page" or "Actual Size" for the best view.
Use the filters on the left (Payment Status, Month, etc.) to interact with the dashboard.
Navigate between Page 1, Page 2, Page 3 at the bottom.


Note: For the best experience, open it in Power BI Desktop (not the web version) to see the full designed layout and interactions.
